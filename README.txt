Mouse drag: create object
Right click while dragging: cancel object creation
Right click on object: delete object or spawner
Left click on object: stop object

0, 1 ... 9: set new object mass
Mouse wheel: set new object mass

q: toggle creation of attractors and repulsors
q on object: toggle repulsor/attractor
z: delete minor objects
x: delete major objects
hold s: create swarm of minor objects
hold s with shift: create bigger swarm of minor objects
hold s with mouse drag: create directed stream of minor objects
Left click while holding s: create permanent minor object spawner
NOTE: creating permanent high-intensity (with shift) spawners is not recommended  
p: toggle pause
g: toggle gravity for major objects
h: toggle gravity for minor objects
